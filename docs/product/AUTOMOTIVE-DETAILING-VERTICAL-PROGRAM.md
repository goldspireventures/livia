## L0 — What Livia means for automotive detailing

Jobs are **2–5 hour bay blocks**, not 30-minute slots. **Vehicle** is continuity object.

**One sentence:** *Livia books detailing packages by bay day — vehicle notes return every visit.*

### Wow — operator

| Moment | Why |
|--------|-----|
| **Bay utilisation** | See long blocks — empty bay = money leak |
| **Vehicle continuity** | Colour, prior damage notes on return |
| **Package tiers** | Bronze/silver/gold clear on book |
| **UK demo locale** | GBP, Belfast market shop |

### Wow — guest

| Moment | Why |
|--------|-----|
| **Package comparison** | Understand what they buy |
| **Drop-off visit page** | Keys, access, ETA |
| **Deposit on premium** | Optional pay token |
| **Premium dark UI** | Matches studio brand |
