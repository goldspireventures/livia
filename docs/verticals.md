## Vertical-specific Livia features (forward references to F4)

- **Hair / Beauty** — colour formula library, allergy patch-test ledger, retail product upsell (none unique).
- **Wellness** — gift-voucher redemption, group-booking (couples massage), spa-day multi-service orchestration.
- **Body art** — design proof + sign-off workflow, deposit-binds-design rule, age-verification gate, healing-aftercare comms cadence.
- **Fitness** — class booking (capacity-bound + recurring), waitlist promotion, package/membership consumption tracking, PARQ intake.
- **Medspa** — informed consent flow, contraindication check, before/after photo handling, complication-protocol escalation. **Not built without medico-legal counsel + insurance review.**
- **Allied health** — progress notes, treatment-plan adherence, GP referral handoff. **Lite version only; will not become a regulated EHR.**
- **Pet grooming** — pet record (breed, allergies, vaccination check, behaviour notes), reactive-cancellation if pet unwell, multi-pet households.
